Solar Shed 0.1

This website is hosted on an off-grid microserver located in a shed with solar power and mobile connectivity.

Hardware

  • Victron Monocrystalline Panel 90W 12V
  • Victron Gel Battery 12V 60Ah
  • Victron MPPT Charge Controller 75V 15A
  • Raspberry Pi Zero W
  • Sixfab 4G/LTE Base HAT
  • Quectel EC25 Mini PCle 4G/LTE Module

Software

  • Debian 11 32-bit
  • Nginx
  • Tailscaled
  • Cloudflared
